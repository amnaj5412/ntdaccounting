# หจก.ณัฏฐธิดา การบัญชี - Static Website

เว็บไซต์เวอร์ชัน static HTML (ไม่มี PHP/MySQL) สร้างจากเนื้อหาปัจจุบันของระบบ CMS
เหมาะสำหรับโฮสต์บน GitHub Pages หรือ static hosting อื่นๆ

## หมายเหตุ
- เนื้อหาทั้งหมด (บริการ, กิจกรรม, บทความ, ข้อมูลติดต่อ) ถูกฝังเป็น HTML คงที่ ณ เวลาที่สร้างไฟล์นี้
- หากมีการแก้ไขเนื้อหาผ่านระบบ Admin ของเว็บไซต์หลัก ต้องสร้างไฟล์ static ชุดนี้ใหม่เพื่อให้ข้อมูลอัปเดต
- ฟอร์มติดต่อในหน้า contact.html เป็น client-side เท่านั้น (ไม่ส่งข้อมูลไปเซิร์ฟเวอร์)
- เปิดหน้าแรกได้ที่ `index.html`
